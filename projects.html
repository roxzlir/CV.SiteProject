Penguin Bank
×

Penguin Bank är ett projektarbete som jag skapat tillsammans med 4 andra klasskompisar. Det var vårt slutprojekt när vi gick kursen Objektorienterad Programmering.

Uppgiften gick ut på att vi skulle skapa en konsol applikation som skulle agera som en bank app. Vi fick utgå efter en krav specifikation/backlog där vi skulle försöka uppfylla så många önskemål som möjligt.

Detta var det första team arbete vi fick köra och jag blev personligen väldigt nöjd med resultatet och hur bra själva samarbetet fungerade!

The School of Programming
×

The School of Programming är ett helt individuellt projektarbete som jag skapat på egenhand till skillnad från Penguin Bank. Detta var slutprojektet på kursen för databaser och SQL.

Här var målet att jag skulle skapa en databas för en skola med hjälp av SQL Server Management Studio efter vissa riktlinjer om vad för data som skulle kunna lagras för denna skola. Sen var nästa steg att skapa en konsol applikation även här men som kunde både läsa och skriva mot min databas. Även här var det vissa krav som skulle uppfyllas och man fick lösa uppgiften med både Entity Framework och även med vanliga SQL frågor för att lösa allt som efterfrågades.

Jag la ganska mycket tid på denna uppgift för jag hade bestämt mig för att jag ville lösa alla krav med Entity Framework. Innan jag va klar så hade jag lyckats med detta och även lagt in lite fler funktioner än vad som önskades!
